```html
<ui-progress></ui-progress>
```

### Props

| Name       | Type    | Default | Description                       |
| ---------- | ------- | ------- | --------------------------------- |
| `active`   | boolean | `false` | 设置进度条为活动状态。            |
| `progress` | number  | `0`     | 设置进度条值。值的范围为 `[0, 1]` |
| `buffer`   | number  | `0`     | 设置缓冲区值。值的范围为 `[0, 1]` |
| `label`    | string  | `''`    | 进度条说明文字。                  |
| `closed`   | boolean | `false` | 隐藏进度条。                      |
