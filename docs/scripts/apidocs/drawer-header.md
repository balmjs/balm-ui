#### Slots

| Name      | Description                                                                     |
| --------- | ------------------------------------------------------------------------------- |
| (default) | The default slot holds the drawer header child components and can contain HTML. |

**Drawer header child components:**

- `<ui-drawer-title>`
- `<ui-drawer-subtitle>`
